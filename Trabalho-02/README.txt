Avisos:

Na classe Menu, quando rodar o programa, não vai ter nenhum membro salvo, por isso, o método login() irá criar um novo membro.
Como o primeiro a acessar este menu, muito provavelmente, será um BigBrother, o membro criado é da classe BigBrother.

Nesse programa, por utilizar lista ligada, os dados são perdidos após finalizar o programa.
Entretanto, como não foi pedido a criação de um banco de dados, decidi usar lista ligada por ser mais fácil de aplicar.
Provavelmente, mais tarde, irei alterar o programa para utilizar um csv ou SQL (O que servir) para resolver esse problema.

Como não estava escrito no enunciado, assumi que a hora de trabalho para cada funcionário é independente da hora de trabalho de outros funcionários.

Artes ASCII foram criadas a partir de um gerador de arte ASCII, link: https://www.ascii-art-generator.org/
